# 💎 GEM Bot (Guild Economy Mint)

GEM（ジェム）は、Discordサーバー内に独自の「経済圏」を創り出し、コミュニティを活性化させるための高機能Discordボットです。
ユーザーは毎日のログインやメンバーとの交流、運営からの付与によって独自の通貨（ポイント）を獲得し、貯めたポイントでサーバー内の限定ロールやアイテムを購入することができます。

また、コミュニティ運営には欠かせないチケットツール機能や、サーバーのエンゲージメントを爆発的に高める「3ヶ月周期のシーズンレベル制度」、コミュニティの定着に特化した「限定パッシブ付リファラル（友達招待）システム」を搭載。

---

## 🌟 主な機能

### 1. 🪙 独自通貨（Economy）システム
* **サーバー専用通貨の発行**: サーバーごとに独自の通貨（例：GEM、SPCなど）を命名し、管理できます。
* **サーバーごとの完全独立設計**: 通貨データはサーバーごとに独立して管理されるため、複数のサーバーで安全に運用可能です。
* **残高確認**: ユーザーは `/gem-balance` でいつでも自分の残高をスマートに確認できます。（他人には見えません）

### 2. 🏪 ギルドストア（Shop）と自動ロール付与
* **商品の販売**: 運営は `/gem-store add` でストアに商品を並べることができます。
* **ロールの自動売買**: ポイントを使って商品を `/buy` すると、紐づけられたDiscordロールが自動的にユーザーへ付与されます。
* **重複購入防止**: 既に持っているロールは購入できない安心設計です。

### 3. 🎲 ログインボーナス & 交流ルーレット
* **自分向けログインボーナス**: 1日1回限定！ボタンを押すだけでランダムなポイントがもらえるログインボーナスを設置できます。（日本時間深夜0時リセット）。毎日連続でログインすると「継続日数」に応じたボーナスを獲得できる**Streak機能**も搭載！
* **他人向けルーレット**: 自分のポイントを消費してルーレットを回し、サーバー内の誰かにランダムなポイントをプレゼント！メンション付きの全体通知でお礼や交流のきっかけを生み出します。

### 4. 📊 GEM レベル ＆ 📅 シーズンシステム（3ヶ月周期）
* **アクティビティによるレベルアップ**: サーバー内での発言（テキストチャット）やリアクション、ログイン継続によって「EXP」を獲得。EXPが貯まると自動で「GEMレベル（1〜10）」が上がり、獲得できる通貨の倍率（ポイントバフ）が永続的に増加します。
* **EXPブースト（バフ）**: 連続ログイン日数（Streak）や、ストアで購入した「特権ロール」によって獲得EXPに倍率がかかる掛け算システムを採用。（※特権ロールの重複はなく、最も高い倍率のものが自動適用されます）
* **限界突破のレベルキャップ**: 現行の獲得ソース（1日5回制限）では、毎日ガチで貢献しても1シーズン（3ヶ月）で「レベル8」あたりが実質的な最終到達ラインとなる、オンラインゲームのようなやり込み要素を含んだ難易度設計になっています。
* **高度なスパム対策**: 連投（クールダウン）、文字数不足、同一メッセージへのリアクション連打などを自動で検知・無効化し、質の高い交流のみを評価します。

### 5. 👥 成果報酬型リファラル（友達招待）システム
* **定着タスク連動型インセンティブ**: 単なる「サーバー入室時」の付与ではなく、招待された新規ユーザーがログインボタンを一定回数押すなどの「定着タスク」を完了した時点で初めて双方に報酬GEMを付与。幽霊部員（即抜け）による不正・ポイント乱獲を100%シャットアウトします。
* **30日間限定パッシブアフィリエイト**: 招待タスク完了後、紹介された側（子）が日々の活動で稼いだEXP/GEMの数%が、紹介した側（親）に最初の30日間限定で自動バック（パッシブ還元）されます。既存ユーザーが新入りのメンター（お世話係）となり、コミュニティに馴染ませる強力な動線を作ります。

## 📅 シーズンサイクル ＆ 自動表彰システム

本システムは、3か月に一度、累積EXPとGEMレベルがリセットされる「シーズン制」で運用されます。レベルそのものをDiscordの「ロール」にせず、Bot内の数値としてオンデマンドで計算・管理するため、動作が非常に軽量です。

【1日 0:00】 シーズン開幕
├ 全ユーザーの「当シーズン獲得EXP」および「GEMレベル」がリセット（通算データは別途保持）
└ 前シーズンのアワードロール保持者に、初期アドバンテージEXPを自動付与

【2ヶ月目・3ヶ月目の1日 12:00】 特大中間発表（Botによる自動Embed投稿）
└ 前月までの熱戦を振り返り、その時点の「獲得GEMランキングトップ10」＆「現在の最高レベル到達者」をアナウンスし、新月のアクティブ率を再点火

【3か月目の月末 23:59】 シーズン閉幕 ＆ 自動表彰式（シーズンアワード）
├ そのシーズンに最も「GEMを稼いだ人」「EXPを稼いだ人」「ルーレットで貢献した人」を自動抽出し表彰
└ 上位入賞者には、次シーズン（3ヶ月間）有効な「特権アドバンテージロール」をBotが自動付与

🏆 特権ロールの命名・アドバンテージルール（例：1stシーズンの場合）

1stシーズン_Platinum : 上位1%に付与 ➔ 次シーズン開幕時に 1,500 EXP 付与（最初からLv.3スタート！）
1stシーズン_Gold    : 上位5%に付与 ➔ 次シーズン開幕時に 1,000 EXP 付与（最初からLv.2後半スタート！）
1stシーズン_Silver  : 上位10%に付与 ➔ 次シーズン開幕時に 500 EXP 付与（最初からLv.2スタート！）
1stシーズン_Bronze  : 上位15%に付与 ➔ 次シーズン開幕時に 100 EXP 付与（最初からLv.1微ブーストスタート！）
