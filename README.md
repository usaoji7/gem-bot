# 💎 GEM Bot (Guild Economy Mint)

GEM（ジェム）は、Discordサーバー内に独自の「経済圏」を創り出し、コミュニティを活性化させるための高機能Discordボットです。
ユーザーは毎日のログインやメンバーとの交流、運営からの付与によって独自の通貨（ポイント）を獲得し、貯めたポイントでサーバー内の限定ロールやアイテムを購入することができます。

---

## 🌟 主な機能

### 1. 🪙 独自通貨（Economy）システム
* **サーバー専用通貨の発行**: サーバーごとに独自の通貨（例：GEM、SPCなど）を命名し、管理できます。
* **サーバーごとの完全独立設計**: 通貨データはサーバーごとに独立して管理されるため、複数のサーバーで安全に運用可能です。
* **残高確認**: ユーザーは `/gem-balance` でいつでも自分の残高をスマートに確認できます。（他人には見えません）

### 2. 🏪 ギルドストア（Shop）と自動ロール付与
* **商品の販売**: 運営は `/gem-store add` でストアに商品を並べることができます。
* **ロールの自動売買**: ポイントを使って商品を `/buy` すると、紐づけられたDiscordロールが自動的にユーザーへ付与されます。
* **重複購入防止**: 既に持っているロールは購入できない安心設計です。

### 3. 🎲 ログインボーナス & 交流ルーレット
* **自分向けログインボーナス**: 1日1回限定！ボタンを押すだけでランダムなポイントがもらえるログインボーナスを設置できます。（日本時間深夜0時リセット）
* **他人向けルーレット**: 自分のポイントを消費してルーレットを回し、サーバー内の誰かにランダムなポイントをプレゼント！メンション付きの全体通知でお礼や交流のきっかけを生み出します。

## 📜 コマンド一覧と使い方

### 👑 運営・管理者向けコマンド
初期設定や、ポイントの付与、ストアの管理を行うコマンドです。（※管理者権限を持つユーザーのみ実行可能）

| コマンド | 引数 | 説明 |
| :--- | :--- | :--- |
| `/gem-setup` | `name` | サーバー独自のオリジナル通貨の名前（単位）を設定します。（初回必須） |
| `/gem-give` | `user`, `amount` | 指定したユーザーにポイントを直接付与します。 |
| `/gem-take` | `user`, `amount` | 指定したユーザーからポイントを剥奪（減算）します。 |
| `/gem-give-role`| `role`, `amount` | 指定したロールを持つメンバー全員に、ポイントを一斉付与します。イベントの報酬などに最適です。 |
| `/gem-store add` | `name`, `price`, `role`(任意), `description`(任意) | ストアに新しい商品を登録します。ロールを指定すると購入時に自動付与されます。 |
| `/gem-store remove`| `item_id` | ストアから指定したIDの商品を削除します。 |
| `/gem-bonus-setup` | `min`, `max`, `target`, `cost`(任意), `title`(任意) | ログインボーナスまたは他人向けルーレットのボタンをチャンネルに設置します。 |
| `/gem-ticket-setup`| `role1`, `role2`(任意), `role3`(任意), `category`(任意), `title`(任意) | お問い合わせやサポート対応用の、プライベートチケット（チャンネル）作成ボタンを設置します。 |

### 👤 一般ユーザー向けコマンド
ユーザーが日常的に利用するコマンドです。（※実行結果は本人にのみ表示され、チャット欄を汚しません）

| コマンド | 説明 |
| :--- | :--- |
| `/gem-balance` | 自分の現在のポイント残高を確認します。 |
| `/store` | 現在ストアで販売されている商品と価格の一覧を表示します。 |
| `/buy [item_id]` | ポイントを消費してストアの商品を購入します。 |
